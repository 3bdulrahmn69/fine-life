# SEO Images Required

For optimal SEO performance, create these images in the `public/` folder:

## Required Images:

### 1. og-image.jpg

- **Dimensions**: 1200x630 pixels (1.91:1 aspect ratio)
- **Purpose**: Open Graph image for social media sharing
- **Content**: Fine Life logo/branding with tagline "Transform Your Financial Future"
- **Format**: JPG, PNG, or WebP

### 2. og-home.jpg

- **Dimensions**: 1200x630 pixels (1.91:1 aspect ratio)
- **Purpose**: Specific Open Graph image for the home page
- **Content**: Hero section style with financial dashboard preview
- **Format**: JPG, PNG, or WebP

### 3. favicon.ico

- **Dimensions**: 32x32, 16x16 pixels (and optionally 48x48, 64x64)
- **Purpose**: Browser tab icon
- **Format**: ICO format

### 4. apple-touch-icon.png

- **Dimensions**: 180x180 pixels
- **Purpose**: iOS home screen icon
- **Format**: PNG

### 5. browserconfig.xml

- **Purpose**: Microsoft tile configuration
- **Content**: Points to msapplication tile images

## Image Guidelines:

- Use high-quality, professional images
- Include Fine Life branding
- Ensure text is readable when scaled
- Optimize file sizes for web (under 500KB each)
- Use consistent color scheme with the app theme

## Tools for Creating Images:

- Canva, Figma, or Adobe Photoshop
- Online OG image generators
- favicon.io for favicon generation
